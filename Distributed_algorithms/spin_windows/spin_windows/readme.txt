Запустите файл spin.bat в каталоге spin_windows


Файл xspin.tcl содержит исходный код GUI XSpin

Данная сборка использует библиотеки ActiveTcl 8.5.10.1 для выполнения кода GUI и MinGW Portable 5.1.4 для компиляции верифицируемой модели.

Spin создает временные файлы в процессе работы, поэтому пользователь, запускающий Spin, должен иметь права на запись в каталог с системой. Также в каталоге не должно быть временных файлов, созданных другим пользователем.
